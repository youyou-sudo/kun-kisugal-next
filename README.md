
# KisuGAL

KisuGAL 是一个免费无门槛的Galgame资源分享平台，为爱发电，让更多人能体验到GalGame的快乐。

本站会不断更新资源库，让网站生态更加丰富！

## 如何运行

跟源项目kun-touchgal-next一样
